# GraphRain — Конструктор графиков с дождём

Интерактивный визуализатор математических функций с необычной физической симуляцией дождя.

## О проекте

**GraphRain** — это не просто построитель графиков. Это полноценный интерактивный конструктор, где на график функции **падает дождь**, который стекает по кривой и собирается в лужи в локальных минимумах.

Проект сочетает в себе математику, интерактивную графику и элементы физической симуляции.

### Основные возможности

- **Построение графиков** пяти различных функций:
  - `sin(x)`
  - `x²`
  - `tan(x)`
  - `x³`
  - `2x + 5`

- **Интерактивность**:
  - Перетаскивание графика (drag & drop)
  - Масштабирование колёсиком мыши
  - Клик по графику — создание «горбов» (гауссовых всплесков)

- **Симуляция дождя**:
  - Реалистичные капли дождя
  - Сток воды по рельефу функции
  - Накопление воды в лужах
  - Взаимодействие с горбами и минимумами

- **Кастомизация**:
  - 6 стилей фона (сплошной цвет, штриховка, шахматка, зигзаг, кирпичи, диагональный градиент)
  - Изменение цвета графика
  - Случайный выбор функции

- **Сохранение** результата в PNG

## Как использовать

1. Запустите приложение.
2. Используйте кнопки в правой панели:
   - **Случайная функция** — сменить график
   - **Стиль фона** — выбрать оформление фона
   - **Стиль графика** — изменить цвет линии
   - **Дождь** — включить/выключить симуляцию
3. **Левая кнопка мыши** — перетаскивание графика
4. **Колёсико мыши** — зум
5. **Клик левой кнопкой** по графику — добавить горб
6. **Сохранить** — экспорт изображения в PNG

## Технологии

- C# (.NET Framework 4.7.2)
- Windows Forms
- GDI+ (Graphics)
- Double Buffering для плавной отрисовки

## Структура проекта

- `Main.cs` — основная форма и вся логика отрисовки
- `AbstFunction.cs` + наследники (`SinFunction`, `TgFunction` и др.) — математические функции
- `DrawBG.cs` — отрисовка фоновых стилей
- `DrawRain.cs`, `WaterPool.cs`, `HumpBack.cs` — классы физической симуляции
- `BackgroundStyle.cs` — форма настройки фона
